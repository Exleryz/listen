### 等级评定

### 做题
- 练习
- 测试

根据用户评定的等级，提供相应难度的题目

一个等级25个关卡

#### 点击关卡

ajax查看历史闯关记录(点击的关卡 分页)

#### 进入做题
题目从管理员设置好的题库池中随机选取，并把选项随机

#### 提交判分
```
超过本关卡规定分数
    且 未超过 关卡总数
        当前关卡 + 1
    若 超过 关卡总数
        当前等级 + 1
        当前关卡数 = 0
没有则继续闯关
```
